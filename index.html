    📖Anleitung für App-Betreiber (einmalig)▼
    ⚠️ Diese Schritte musst nur du einmalig machen. Alle anderen klicken danach einfach «Mit Google anmelden».
    1
    🐙 App auf GitHub Pages hosten
    • 1Gehe zu github.com → Account erstellen (kostenlos)
    • 2Oben rechts auf «+» → «New repository» · Name: school-planer · Visibility: Public → «Create repository»
    • 3«uploading an existing file» → SchulPlaner.html hochladen → «Commit changes»
    • 4«Settings» → «Pages» → Branch: main → «Save»
    • 5✅ Nach 2 Min erreichbar: https://DEINNAME.github.io/school-planer/SchulPlaner.html
    2
    🔵 Google Cloud Console: Projekt erstellen
    • 1Gehe zu console.cloud.google.com
    • 2Oben links: «Projekt auswählen» → «Neues Projekt» → Name: School-Planer → «Erstellen»
    • 3Linkes Menü: «APIs & Dienste» → «Bibliothek» → «Google Drive API» suchen → «Aktivieren»
    3
    📋 OAuth Zustimmungsbildschirm konfigurieren
    • 1«APIs & Dienste» → «OAuth-Zustimmungsbildschirm» → Benutzertyp: Extern → «Erstellen»
    • 2App-Name: School-Planer · Support-E-Mail: deine Gmail · Autorisierte Domain: github.io
    • 3Unter «Scopes»: «Scopes hinzufügen» → drive.appdata suchen und hinzufügen → Speichern
    • 4«Speichern und fortfahren» durch alle Schritte → «Zurück zum Dashboard»
    • 5«App veröffentlichen» → «Bestätigen» (damit alle, nicht nur Testnutzer, sich anmelden können)
    4
    🔑 Client ID erstellen & eintragen
    • 1«APIs & Dienste» → «Anmeldedaten» → «Anmeldedaten erstellen» → «OAuth-Client-ID»
    • 2Anwendungstyp: Webanwendung · Name: School-Planer
    • 3«Autorisierte JavaScript-Quellen» → https://DEINNAME.github.io hinzufügen
    • 4«Erstellen» → Client ID kopieren (sieht aus wie: 123456.apps.googleusercontent.com)
    • 5Client ID oben ins Feld «Google Client ID» einfügen → 💾 speichern
    • 6✅ HTML-Datei erneut auf GitHub hochladen → alle können «Mit Google anmelden» nutzen!
